# `/value-board-summary` 示例文档（虚构）

> 本文档演示 `/value-board-summary` 命令的输出形态。所用公司 **「晨曦咖啡（Sunrise Coffee）」是完全虚构的示例**，所有数字均为**标注的假设（Assumption）**，仅用于教学。
>
> **这不是投资建议、不含目标价、不含买卖判断。** This is a research and learning framework, not an investment recommendation.

`/value-board-summary` 退出角色扮演，把整场董事会研究记录汇总成一份**可追溯**的结论。它固定六个部分，每条结论都标注证据类别（`Fact` / `Management statement` / `Inference` / `Assumption` / `Unknown`）。模板见 `references/evidence-ledger.md`。

## 二、会话问答节选（briefing，角色回合如何引出驱动因素）

> 每个回合遵循契约：陈述事实/假设 → 暴露激励冲突 → 画出 FCF 路径 → 只问一个问题。

**`[cfo]`（财务总监）**
- 事实/假设：本店客单价约 ¥25/杯（`Assumption`）；门店层面毛利率约 65%（`Assumption`）。
- 冲突：市场部要加品牌投入，我（CFO）要 ROI 证据。
- FCF 路径：客单价 × 杯数 → 门店收入 → 扣变动成本得毛利 → 扣租金等固定成本得营业利润。
- 问题：*"每新开一家店的建店投资，多久能从该店现金流回本？"*

**`[operations]`（运营/门店）**
- 事实/假设：单店建店投资约 ¥80 万（`Assumption`）；假设回本期约 24 个月（`Assumption`）。
- 冲突：要冲规模就要多开店，但租金刚性且逐年上涨。
- FCF 路径：新店 CapEx → 回本后变为自由现金流；维护性 CapEx（设备翻新）约占收入 2–3%（`Assumption`）。
- 问题：*"租金占门店收入的比例，在新签租约时是上升还是下降？"*

**`[marketing]`（市场/品牌）**
- 事实/假设：停投广告 12 个月后，复购率假设先于新客下降（`Assumption`）。
- 冲突：品牌投入难短期量化，却影响长期定价能力。
- FCF 路径：品牌 → 复购率 → LTV → 降低获客成本 → 改善 FCF。
- 问题：*"如果不打广告，哪个指标先掉、掉多少？"*

这三轮问答直接指向了下面的三个核心 FCF 驱动因素。

## 三、董事会结论（`/value-board-summary` 的六段输出）

### 1. 客户付款 → 自由现金流的机制

晨曦咖啡把客户付款变成自由现金流的链条（全部为 `Assumption`）：

```
客单价(¥25) × 杯数 × 客群结构(堂食/外带/零售豆)
   → 门店收入
      − 变动成本(豆、奶、包装、时薪人工，约 35%)
      − 固定成本(租金为主，另含折旧、总部)
         → 营业利润
            − 营运资本投入(应收≈0，因现结；存货低；应付=租金/供应商账期)
               → 经营性现金流
                  − 维护性 CapEx(设备翻新)
                  − 增长性 CapEx(新店建店)
                     → 自由现金流
```

关键特征（`Inference`）：现款现结使应收账款近乎为零，叠加低存货，形成**轻营运资本甚至负营运资本**——客户付款当天到账，而租金/货款有账期。这是现金转化上的结构性优势；真正的现金消耗在**增长性 CapEx（开新店）**。

### 2. 5 / 10 / 20 年：稳定 vs 变化

| 期限 | 倾向稳定的 | 可能变化的 |
|---|---|---|
| 5 年 | 现款现结的现金流结构、负营运资本优势（`Inference`） | 单店回本期（受租金与杯量影响，`Unknown`） |
| 10 年 | 咖啡消费习惯、门店模型的单位经济（`Assumption`） | 市场饱和度与同店增长（`Unknown`）；租金成本曲线（`Unknown`） |
| 20 年 | 品牌/选址形成的轻壁垒（若成立）（`Assumption`） | 消费代际偏好、自动化对人工成本的影响、竞争格局（`Unknown`） |

### 3. 熊 / 基准 / 牛（条件式，非精确预测）

| 情景 | 触发条件 | 传导路径 | FCF 方向 | 证伪指标 |
|---|---|---|---|---|
| **熊** | 租金通胀 + 区域饱和 | 同店杯量转负 → 固定成本（租金）摊薄变差 → 新店回本拉长至 >3 年 → 增长 CapEx 烧钱 | 下降 | 同店销售连续两季为负；新店回本期延长 |
| **基准** | 同店中个位数增长，回本 ~24 个月 | 杯量温和增 → 营业杠杆 → 新店按期回本 → FCF 随门店数增长 | 温和增长 | 同店增长维持中个位数；单店回本 ≤30 个月 |
| **牛** | 定价能力 + 网络扩张 + 负营运资本放大 | 提价带动毛利 → 新店快速回本 → 经营现金先于利润到账 → FCF 复利 | 加速增长 | 客单价提升且杯量不减；新店回本 ≤18 个月 |

> 三种情景都是**带条件的故事**，不是单一伪精确预测。每条都附会推翻它的经营指标。

### 4. 最多三个 FCF 驱动因素

1. **同店杯量 × 客单价（同店销售）**——租金是固定成本，杯量直接决定营业杠杆，是利润和现金流的最大杠杆点。机制：杯量 ↑ → 租金占比 ↓ → 营业利润率 ↑。（`Inference`）
2. **新店回本期 / 单位经济**——决定增长性 CapEx 是创造现金还是消耗现金。机制：回本短 → 每家店更快转为正自由现金流。（`Assumption`）
3. **租金占收入比**——固定成本摊薄的边界，也是熊情景的主因。机制：租金占比 ↑ → 单店营业利润 ↓ → 回本期拉长。（`Assumption`）

### 5. 反证与未知（按重要性排序）

- `Unknown`：**市场饱和度与同店增长的真实数据**——会直接证伪基准/牛情景，最需优先取得。
- `Unknown`：**区域租金趋势与新签租约条款**——决定固定成本曲线。
- `Unknown`：**新店之间的相互蚕食（cannibalization）**——网络扩张时单店杯量是否被稀释。
- `Unknown`：**人工成本法规（最低工资、用工方式）**——影响变动成本。
- `Assumption`→待证伪：24 个月回本的假设；若实际 >30 个月则基准情景动摇。
