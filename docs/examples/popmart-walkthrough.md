# 泡泡玛特（Pop Mart）董事会演练示例

> 本文档演示 `value-boardroom` 技能套件在真实公司的实际效果。**这是一份研究演练，不是投资建议。**

公司：**泡泡玛特（Pop Mart International Group Limited，HKEX: 9992）**

数据来源：Pop Mart 2024 年年度业绩（通过 Yahoo Finance 和公司公告提炼）、Wikipedia（2026-06-19 复核）、公司官方公告。

**证据类别速查：** `Fact` = 有来源的事実（附日期）；`Management statement` = 公司声明（未独立核实）；`Inference` = 由事实推导的经营结论；`Assumption` = 可证伪的模型输入；`Unknown` = 缺可靠来源的关键信息。

## 2. `/value-role-ceo`（CEO —— 增长来源、战略边界、资本配置）

> briefing 模式：模型模拟 CEO，用户作为董事发问。

**主张（Assumption → 待核实）：** 泡泡玛特的增长由三股力量驱动: IP 热度（尤其是 THE MONSTERS/Labubu 2024 年的全球爆发）、海外扩展（东南亚 → 全球），以及盲盒 + 毛绒品类的品类扩张。

- **IP 热度**（`Fact`）：THE MONSTERS（Labubu）在 FY2024 成为 IP 收入的绝对主力，毛绒品类贡献了可观份额（来源：公司 2024 年报）。自研 IP 原占收入约 76.5%（2023 年，Wikipedia）。
- **海外扩展**：海外门店从 2022 年初的 18 家快速扩张；至 2025 年 6 月仅美国已 37 家门店 + 52 台机器人商店（Wikipedia）。海外已成为"最重要的发展重点"（公司副总裁，引用自 FT 2022）。
- **品类扩张**：从盲盒拓展到毛绒玩具（Labubu plush），单价更高 -> 拉动平均客价。（`Inference`）

**冲突（Assumption）：** 增长跟 IP 生命周期绑定。Labubu 2024 年的爆发性热度若衰减，同店收入将急剧承压——这是增长速度和可持续性之间的经典张力。

**FCF 路径：** IP 热度 → 杯数（盲盒/毛绒售出件数）→ 平均售价（RMB 59–69/盒）→ 门店/渠道收入 → 营业利润 → 自由现金流。

**董事问题：** *"如果 Labubu 的全球热度在未来 2 年内减退到 Molly 2021 年的水平，单店收入会下降多少？公司有什么 IP 管线对冲？"*

## 3. `/value-role-cfo`（CFO —— 利润质量、杠杆、营运资本、资本配置）

**关键经营事实（`Fact`，来源：Yahoo Finance TTM 截至约 2026 年 4 月）：**

| 指标 | TTM 值（HKD） | 约合 RMB（1 HKD ≈ 0.92 RMB） |
|---|---|---|
| 收入（TTM） | 371.2 亿 HKD | ~~340 亿 RMB（FY2024 全年） |
| 净利润（TTM） | 127.8 亿 HKD | ~~117 亿 RMB |
| 利润率 | 34.4% | — |
| 杠杆 FCF（TTM） | 91.1 亿 HKD | ~~84 亿 RMB |
| 现金（MRQ） | 172.3 亿 HKD | ~~159 亿 RMB |
| 负债/权益 | 12.6% | — |
| ROE（TTM） | 77.6% | — |
| P/E（TTM） | 13.8x | — |

**FCF 质量分析（`Inference`）：**
- 高利润率（34% 净利率）和高 ROE（78%）表明定价能力 + 轻资产模式。
- 负债极低（D/E 12.6%），现金充裕——资金压力低。
- FCF/净利润 ≈ 71%（91.1/127.8），意味着大约 29% 的利润用于营运资本投入和 CapEx。需要进一步拆分维护性 CapEx vs 增长性 CapEx（`Unknown`——年报细节待查）。

**冲突：** 利润质量与增长 CapEx 之间的权衡——海外扩张是新开店投资，若新店回本期延长，FCF 转化率会恶化。

**董事问题：** *"TM 的自由现金流里，有多少用于增长性 CapEx（新店 + 新模具），多少是维持现有业务的？新店从投资到回本大约需要多长时间？"*

## 4. `/value-cashflow`

**客户付款 -> FCF 的完整桥梁（主要基于 Yahoo Finance TTM 数据，标注 `Fact` / `Inference` / `Assumption`）：**

```
平均售价 RMB 59–69/盒 × 售出件数 × IP 组合
   → 收入 FY2024 约 RMB 340 亿（Fact）
      − 变动成本(COGS — 玩具制造成本、版权/设计分成、包装)
      − 固定成本(门店租金、员工、折旧)
         → 营业利润（Inference — 具体毛利率未公开，但净利 34% 指指向较高毛利）
            − 营运资本投入(应收账款极低——零售现结；存货=玩具 + 原材料；应付=供应商账期)
               → 经营性现金流 OCF
                  − 维护性 CapEx（模具更新、门店翻新）
                  − 增长性 CapEx（新开店、机器人商店、海外扩展）
                     → FCF ≈ HKD 91 亿 TTM（Fact）
                        → 资本配置：海外开店 × 现金储备 HKD 172 亿 ÷ 分红
```

**存货周转特点（`Inference`）：** 盲盒的"隐藏款"机制创造人为稀缺性 → 提高卖出件数的同时要求更多 SKU 库存。存货管理是关键——过季 IP 的尾货减值风险。

**ROIC 初步框架（`Assumption`）：** ROE 77.6% 是极高值，部分来源于高利润率 × 轻资产，部分来源于杠杆（D/E 12.6% 很低，杠杆贡献不大）。ROIC 应接近 ROE（负债极少）。关键是：新增投入资本的回报能否维持——如果增长 CapEx（新店）的 ROIC 低于存量资产的 ROIC，未来 FCF 增长会减速。

**最多三个 FCF 驱动因素（`Inference`）：**
1. **IP 热度 → 售出件数 × 平均价格**——单店杯量决定了固定成本（租金）的摊薄，是经营杠杆的核心。机制：IP 热度↑ → 件数↑ → 租金占比↓ → 营业利润↑ → FCF↑。
2. **海外扩张的 ROIC**——新开海外店的增量投入资本回报是否 ≥ 国内现有门店。机制：海外回本期越短 → 增长 CapEx 效率越高 → FCF 转化率越高。
3. **毛绒品类占比**——毛绒玩具单价 > 盲盒，若品类占比提升 → 客单价↑ → 毛利↑（假设毛绒毛利率 ≥ 盲盒）。机制：品类结构↑ → 平均售价↑ → 增量利润。

## 5. `/stress-test 客单价下降10%`

**冲击：** 假设核心盲盒客单价从平均 ¥65 降至 ¥58.5（-10%），由竞争加剧或消费者支出谨慎驱动（完全虚构假设）。

**传导路径（条件式故事，非精确预测）：**

```
价格 -10%
   → 杯量？(需观察：价格弹性未知，Unknown）
      → 收入仅降 10% ÷ 杯量对冲
         → 变动成本随杯量同步下降(蓝本)
            → 固定成本（租金）不变 → 营业杠杆恶化
               → 营运资本：若杯量下降，存货积压 → 营运资本占用增加
                  → CapEx：门店扩张计划可能收缩
                     → FCF：乐观估计降 15-20%，悲观超 30%
```

**熊 / 基准 / 牛（条件式，附证伪条件）：**

| 情景 | 触发 | FCF 方向 | 证伪指标 |
|---|---|---|---|
| **熊** | 降价 +10% 杯量不升反降（= 品牌力弱） | FCF 大幅下滑 | 同店杯量连续季降；存货周转天数上升 |
| **基准** | 降价后杯量升 5%，部分抵消单价下降 | FCF 温和下降 10-15% | 杯量对冲率 <100%；毛利率稳定 |
| **牛** | 降价刺激杯量升 15%+，规模摊薄租金 | FCF 几乎不受影响 | 杯量升幅 ≥ 单价降幅；新客留存率维持 |

**管理应对（假设）：** 如降价成立，管理层可能（1）收缩增长 CapEx（海外开店减速）；(2) 加速 IP 管线推新以恢复客价；(3) 关闭低效门店。（`Assumption`）

## 6. `/kill 2030`（预演破产）

**假设：** 泡泡玛特在 2030 年发生现金流危机——无法自给现有业务，被迫压缩投资或过度举债。

**倒推最可能路径：**

1. **起点（`Inference`）：** Labubu 热度在 2026-2027 年达到峰值后衰减，全球需求回归长期趋势线。
2. **传导：** 现金最多的 IP 进入下行 → 同店杯量下滑 → 收入增长骤降。同时，为维持增长率而加速海外开店，消耗大量增长 CapEx。
3. **放大器：** 海外新店的回本期 > 预期（文化差异 / 品牌渗透慢），增长 CapEx 不断烧钱却未转为正 FCF。
4. **关键节点 2028-2029：** 增量投入资本回报低于资本成本——公司开始用 FCF 补贴增长，实际上"烧现金买增长"。
5. **2030：** FCF 不足以覆盖维护性 CapEx + 现有库存的减值，公司面临现金紧张。

**最早警示指标（`Inference`）：** 海外同店杯量连续 3 季下滑 + 新店回本期拉长至 >3 年。

**董事决策问题：** *"如果未来两年新店回本期从假设的 24 个月恶化到 >36 个月，公司应该继续开店——还是暂停扩张，把现金用于股份回购 / 还债 / 提高分红？"*

## 7. `/value-board-summary`

### 1. FCF 机制

泡泡玛特通过 IP 驱动的盲盒 / 毛绒玩具零售运转：客户以 RMB 59-69 购买限量/随机玩具 → 现款现结（轻/负营运资本）→ 高利润率（净利 ~34%）→ 扣除开店 CapEx 后产生丰厚 FCF（TTM HKD 91 亿）。ROE 极高（78%），反映轻资产模式 + IP 定价权。

### 2. 5 / 10 / 20 年：稳定 vs 变化

| 期限 | 倾向稳定的 | 可能变化的 |
|---|---|---|
| 5 年 | 盲盒的收集机制驱动复购；现款现结的现金流结构（`Inference`） | Labubu 单 IP 的热度曲线；海外回本期（`Unknown`） |
| 10 年 | 设计师玩具品类存在；IP 矩阵化能力（`Assumption`） | 中国消费偏好变迁；监管风险（盲盒限制已在收紧）（`Fact`） |
| 20 年 | 娱乐/收藏品行业存在（`Inference`） | 技术 / 娱乐平台颠覆玩具品类路径几乎不可预测（`Unknown`） |

### 3. 熊 / 基准 / 牛

| 情景 | 触发 | FCF 方向 | 证伪 |
|---|---|---|---|
| **熊** | IP 热度衰减 + 海外开店低效 + 监管收紧 | 大幅下降 | 同店杯量、海外回本期、监管政策 |
| **基准** | IP 矩阵稳定轮转 + 海外稳步扩展 + 维持毛利 | 温和增长 | 杯量维持、ROIC 稳定 |
| **牛** | 后 Labubu 时代仍有新 IP 爆发 + 海外 ROIC = 国内 + 品类升级 | 高增 | 新 IP 销售贡献；海外 ROIC |

### 4. 最多三个 FCF 驱动

1. **IP 热度 → 单店杯量 × 客单价** —— 固定成本（租金）摊薄的核心。
2. **海外扩展的 ROIC** —— 增长 CapEx 是创造价值还是消耗价值。
3. **毛绒 / 高端品类占比** —— 结构提价 = 无额外 CapEx 的增量利润。

### 5. 反证与未知

- `Unknown`：Labubu 热度的全球衰减路径
- `Unknown`：海外新店的真实回本数据（分地区）
- `Unknown`：IP 管线的储备深度——后 Labubu 时代谁是下一个 Labubu？
- `Unknown`：中国盲盒监管的收紧节奏——已设 CNY 200 价格上限 & 禁向 8 岁以下销售（`Fact`，引用自新华社/Reuters 2023），进一步限制的影响未知
