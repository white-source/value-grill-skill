# Value Boardroom Skills（مهارات مجلس الإدارة الاستثماري）

> [English](README.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [Español](README.es.md) | [Русский](README.ru.md) | [한국어](README.ko.md) | **العربية**

مكتبة Agent Skills محمولة تساعد المستثمر على دراسة شركة مدرجة **كمالك ومشغّل**. تستخدم حوارات سقراطية قائمة على الأدوار بأسلوب مجلس الإدارة لربط القرارات التشغيلية بالتدفق النقدي الحر المستقبلي (FCF) — بدلاً من إنتاج تقارير استثمارية عامة أو توصيات تداول.

## الهدف

- فهم كيف **تخلق** الشركة النقد **وتستهلكه وتوزعه**.
- التفكير كعضو مجلس إدارة أو مالك مسيطر، وليس كمضارب قصير الأجل.
- فصل الحقائق المُتحقق منها عن تصريحات الإدارة، والاستنتاجات، والافتراضات النموذجية، والمجهول.
- تحويل رؤى الأقسام إلى مجموعة صغيرة قابلة للاختبار من محركات FCF والسيناريوهات والأدلة المضادة.

## التثبيت

طريقتان للتثبيت، فلسفتان:

- **Skills CLI** تنسخ المهارات إلى مشروعك لتتمكن من تعديلها وجعلها خاصة بك.
- **إضافة Claude Code** تُبقيها كحزمة مُدارة، للقراءة فقط، ومحدّثة دائماً — تكتفي بالاشتراك بدلاً من التفرع (fork).

### الخيار 1 — Skills CLI (قابل للتعديل، يعمل مع أي بيئة Agent Skills)

```bash
npx skills@latest add white-source/value-grill-skill
```

يعمل مع Claude Code و Codex و Cursor و Copilot وغيرها من البيئات المتوافقة مع Agent Skills. تُنسخ الملفات إلى مستودعك وأنت حر في تعديلها.

### الخيار 2 — إضافة Claude Code (مُدارة، جاهزة فوراً)

تفضّل تثبيتاً جاهزاً لا تحتاج لصيانته يدوياً؟ تتوفر مجموعة المهارات هذه أيضاً كإضافة أصلية لـ Claude Code. بدلاً من نسخ ملفات قابلة للتعديل، تُثبّت الإضافة المجموعة كاملةً كحزمة مُدارة تُحدّث تلقائياً عند إصدار نسخة جديدة.

داخل Claude Code:

```
/plugin marketplace add white-source/value-grill-skill
/plugin install value-grill-skill@white-source
```

أو من سطر الأوامر:

```bash
claude plugin marketplace add white-source/value-grill-skill
claude plugin install value-grill-skill@white-source
```

لا حاجة لأي خطوة إعداد منفصلة — ابدأ الجلسة فوراً بـ `/value-boardroom [الشركة]`.

## البداية

```
/value-boardroom [اسم الشركة أو رمزها]
```

عند البدء تؤكد الجلسة: الشركة (ومعرفها/سوقها إن عُرف)، التاريخ المرجعي، هدف البحث، وضع الأدلة (مقدمة من المستخدم فقط، أو بحث ويب عام عندما يستطيع الوكيل الوصول)، ووضع الدور.

## عكس الأدوار — الآلية الأساسية

الميزة الأبرز لهذه المكتبة هي **عكس الأدوار**: يمكنك دراسة الشركة من أيٍّ من جانبي طاولة مجلس الإدارة، وتعمل الأدوار التشغيلية الأربعة عشر نفسها في كلا الاتجاهين.

| الوضع | النموذج يلعب دور | أنت تلعب دور | لماذا تستخدمه |
|---|---|---|---|
| `briefing` (افتراضي) | المدير التنفيذي المُختار — بحوافزه وميزانياته وتضارباته | مدير مجلس إدارة / شريك يطرح الأسئلة | تبني الفهم باستجواب التفاصيل التشغيلية الحقيقية؛ يجب على المدير التنفيذي تبرير كل رقم وصولاً إلى التدفق النقدي |
| `examination` | مجلس الإدارة | المدير التنفيذي أو المدير المالي أو المدير المُختار — تجيب | يختبر ما إذا كنت **أنت** تفهم العمل فعلاً؛ يلتقط المجلس الثغرات والعموميات في استدلالك أنت |

**briefing** = تتعلم بالاستجواب. **examination** = تتعلم بأن تُستجوب. اختر الوضع بسؤال واحد سريع عند البدء (افتراضي `briefing`) وبدّله في أي وقت بـ `/value-mode` — أو قل ببساطة ("اعكس الأدوار"، "اختبرني"). لا حاجة لإعادة التشغيل. في كلتا الحالتين، عقد الدورة واحد: اذكر حقيقة أو افتراضاً مُعلّماً، واكشف تضارب حوافز واحداً، وارسم مساراً واضحاً للتدفق النقدي، واطرح سؤالاً واحداً بالضبط.

## الأدوار التشغيلية

أربعة عشر دوراً تغطي الشركة بالكامل. ادخل إلى أي دور باستخدام الاختصار ذاتي الإكمال `/value-role-<الدور>` (مثال `/value-role-cfo`)، أو اربط عدة أدوار بـ `/value-role ceo cfo ...`، أو تنقل خطوة بخطوة بـ `/value-next`. كل دور يتحدث فقط ضمن نطاقه، ويكشف تضارب ميزانية أو استراتيجية، ويربط نقطته بالإيرادات أو التكاليف أو رأس المال العامل أو CapEx أو تخصيص رأس المال.

| الدور | ما يملكه | ما تستجوبه |
|---|---|---|
| `secretary` | أمين سر المجلس / المدرب — جدول الأعمال، شجرة الأسئلة، الأدلة، الافتراضات | هل كل استنتاج له مصدر وقابل للتتبع |
| `ceo` | الرئيس التنفيذي — نموذج العمل، مصادر النمو، الاستراتيجية، الثقافة المؤسسية، تخصيص رأس المال | قوة نموذج العمل الدفاعية (إغلاق العميل، تكلفة التحويل، تأثيرات الشبكة)، النمو العضوي مقابل المكتسب، معدل إعادة الاستثمار، توافق الثقافة والمواهب، العائد على رأس المال |
| `cfo` | المدير المالي — جودة الربح، الرافعة المالية، رأس المال العامل، التخصيص | دورة تحويل النقد، الربح المحاسبي مقابل النقدي، CapEx الصيانة مقابل النمو |
| `product` | المنتج / البحث والتطوير — مشكلة العميل، الخارطة، خطر الاستبدال | القوة التسعيرية، الاحتفاظ/التسرب، كفاءة البحث والتطوير |
| `marketing` | التسويق / العلامة التجارية — الاكتساب، الولاء، الاعتماد على الإعلان | CAC، LTV، ما الذي ينخفض أولاً إذا توقف الإعلان |
| `sales` | المبيعات — جودة الطلب، التسعير، التركّز | السعر المحقق بعد الخصومات، تركّز العملاء، الذمم المدينة |
| `channel` | القناة — مخزون الموزعين، الترويج، الحسومات | أيام المخزون في القناة، معدل البيع، هامش القناة |
| `supply` | سلسلة التوريد / المشتريات — تكاليف المدخلات، القوة التفاوضية | تركّز الموردين في تكلفة البضاعة، شروط العقود، فترات التوريد |
| `operations` | العمليات / التصنيع — الاستخدام، العائد، تكلفة الوحدة | معدل الاستخدام، امتصاص التكاليف الثابتة، CapEx التوسع |
| `customer-success` | نجاح العميل / ما بعد البيع — الاحتفاظ، التسرب، التجديدات | الاحتفاظ الإجمالي/الصافي، التوسع مقابل الانكماش، تكلفة الخدمة |
| `people` | الموارد البشرية / المؤسسة — المواهب الحرجة، الحوافز، التعاقب | معدل الدوران في المناصب الرئيسية، هيكل الحوافز، الاختناقات |
| `legal-risk` | القانوني / الامتثال / المخاطر — التراخيص، الدعاوى، التنظيم | الالتزامات الطارئة، التعرض التنظيمي، حوكمة البيانات |
| `strategy-ma` | الاستراتيجية / الاندماجات والاستحواذ — خطوط أعمال جديدة، التوسع، الاستحواذ | العوائد بعد الاستحواذ، الشهرة، تكلفة التكامل |
| `ir-pr` | علاقات المستثمرين / العلاقات العامة — الالتزامات، الشفافية، السمعة | التوجيهات العامة مقابل الفعلية، السرد مقابل العمليات |

## خريطة الأوامر

| الأمر | ما يفعله |
|---|---|
| `/value-ask [سؤال أو هدف]` | صف ما تريد؛ تحصل على الأمر المناسب مع شرح |
| `/value-boardroom [الشركة]` | تهيئة أو إعادة ضبط جلسة مجلس الإدارة |
| `/value-mode [briefing\|examination]` | تبديل وضع الدور في أي وقت — أو فقط قل ذلك |
| `/value-role-<الدور>` | اختصار ذاتي الإكمال لدور واحد — اكتب `/value-role-` واختر (مثال `/value-role-cfo`) |
| `/value-role [دور...]` | ربط عدة مدراء تنفيذيين بالتسلسل، مثال `/value-role ceo cfo supply` |
| `/value-next` | الانتقال إلى الدور التالي في الجولة المقترحة — دون إعادة كتابة |
| `/value-research [موضوع]` | جمع أو طلب معلومات عامة قابلة للتحقق وتصنيفها حسب فئة الدليل |
| `/value-assumption` | عرض أو تعديل الافتراضات النموذجية الجوهرية |
| `/value-evidence` | عرض سجل الأدلة |
| `/value-cashflow` | ترجمة النتائج التشغيلية إلى إيرادات وهامش ورأس مال عامل وCapEx وFCF وROIC |
| `/value-stress-test [صدمة]` | تشغيل صدمة مُسمّاة أو بناء سيناريوهات هبوطية/أساسية/صعودية عبر سلسلة التدفق النقدي |
| `/value-kill [سنة]` | تشريح استباقي: افترض فشل التدفق النقدي بحلول السنة المختارة واعمل عكسياً |
| `/value-board-summary` | الخروج من تمثيل الأدوار وتجميع استنتاج مجلس إدارة قابل للتتبع |

## مثال: Pop Mart (泡泡瑪特, HKEX: 9992)

> ليست توصية استثمارية. كل ادعاء يحمل فئة دليل. [الجولة الكاملة (بالصينية) →](docs/examples/popmart-walkthrough.md)

عرض موجز على شركة مدرجة حقيقية بأرقام مالية حقيقية (Yahoo Finance TTM، Wikipedia).

```text
> /value-boardroom Pop Mart --mode briefing
> /value-role-cfo
```
| المؤشر | TTM (HKD) | الفئة |
|---|---|---|
| الإيرادات | 37.1 مليار | `Fact` |
| صافي الربح | 12.8 مليار (34%) | `Fact` |
| FCF | 9.1 مليار | `Fact` |
| النقد | 17.2 مليار (دين/حقوق 12.6%) | `Fact` |

```text
> /value-cashflow
```
¥59–69/صندوق → إيرادات ¥340 مليار → هامش مرتفع → رأس مال عامل خفيف → CapEx → **FCF HKD 9.1 مليار**. ثلاثة محركات: حركة المتاجر المماثلة، ROIC الخارجي، مزيج الفئات الممتازة.

```text
> /value-stress-test السعر -10% / > /value-kill 2030 / > /value-board-summary
```
سلسلة انتقال الصدمة + التشريح الاستباقي + استنتاج المجلس من 6 أقسام — الكل **كقصص شرطية بمعايير تفنيد**، وليس تنبؤاً وحيداً بدقة زائفة. *بدون توصيات أو أسعار مستهدفة أو تعليمات تداول.*

## فئات الأدلة

كل ادعاء يُوسم لتبقى الاستنتاجات قابلة للتتبع:

| الفئة | المعنى |
|---|---|
| `Fact` | ادعاء مدعوم بمصدر، متضمناً تاريخ النشر والمصدر |
| `Management statement` | تصريح صادر عن الشركة لم يتم التحقق منه بشكل مستقل |
| `Inference` | استنتاج تشغيلي مُشتق من حقائق مُستشهد بها |
| `Assumption` | مدخل نموذجي يمكن تفنيده |
| `Unknown` | معلومات جوهرية بدون دعم موثوق |

## البحث عبر الويب

يُستخدم بحث الويب العام عند توفره مع ذكر المصدر والتاريخ. عندما لا تتوفر أدوات البحث، **تتدهور المهارات بأمانة**: تذكر الأدلة المفقودة، وتسمي المصادر الأولية الواجب الرجوع إليها، وتقترح عبارات بحث، وتطرح سؤال المجلس التالي. لا تختلق أبداً بيانات أو مصادر أو صلاحية وصول أو تحققاً.

## حدود الأمان

هذا إطار بحث وتعلّم. وهو **لا** يقدم توصيات استثمارية أو أسعاراً مستهدفة أو إشارات شراء/بيع/احتفاظ أو تعليمات تداول.
