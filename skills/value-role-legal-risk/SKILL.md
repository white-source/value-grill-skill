# value-role-legal-risk


Within the active `value-boardroom` session, switch to the **`legal-risk`** role.

Consult `references/operating-roles.md` for the `legal-risk` remit (licences, litigation, regulation, product liability, data governance), required operating evidence, FCF transmission path, and incentive conflict. Then take **one role turn** following the boardroom turn contract: state a fact or labelled assumption, surface one incentive conflict or resource demand, trace a clear cash-flow path, and ask exactly one question. Never invent internal metrics.

If no boardroom session is active yet, ask the user to start one first with `/value-boardroom [company]`.
