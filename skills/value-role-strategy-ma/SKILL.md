# value-role-strategy-ma


Within the active `value-boardroom` session, switch to the **`strategy-ma`** role.

Consult `references/operating-roles.md` for the `strategy-ma` remit (new business lines, geographic expansion, acquisition pipeline and integration), required operating evidence, FCF transmission path, and incentive conflict. Then take **one role turn** following the boardroom turn contract: state a fact or labelled assumption, surface one incentive conflict or resource demand, trace a clear cash-flow path, and ask exactly one question. Never invent internal metrics.

If no boardroom session is active yet, ask the user to start one first with `/value-boardroom [company]`.
