# value-role-ceo


Within the active `value-boardroom` session, switch to the **`ceo`** role.

Consult `references/operating-roles.md` for the `ceo` remit (business model analysis, growth sources, strategic boundaries, corporate culture, capital allocation), required operating evidence (business model defensibility, growth split, market share, culture/talent signals, reinvestment rate), FCF transmission path, and incentive conflict. Then take **one role turn** following the boardroom turn contract: state a fact or labelled assumption, surface one incentive conflict or resource demand, trace a clear cash-flow path, and ask exactly one question. Never invent internal metrics.

If no boardroom session is active yet, ask the user to start one first with `/value-boardroom [company]`.
